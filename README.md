# bsky to witch

A browser extension that automatically redirects `bsky.app` to `witchsky.app`.

## Install

This won't be published on the Chrome Web Store or Firefox Add-ons.

- Clone the repository, or download it as ZIP and unzip it
- Turn on Developer Mode in your browser (should be where you manage extensions)
- Load it as unpacked extension, select the `bsky-to-witch` folder
